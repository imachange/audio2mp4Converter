# Audio2MP4 Converter

音声ファイルと画像ファイルを自動的に結合し、YouTubeやSNSへのアップロード用にMP4動画を作成するツールです。
フォルダ内に素材を置いて実行するだけで、FFmpegを使用して動画を生成します。

## 🚀 特徴

* **マルチプラットフォーム**: Windows, macOS, Linux すべての環境で動作します。
* **自動検出**: ファイル名が `audio`（音声）と `cover`（画像）であれば、拡張子を問わず自動で認識します。
* **サイズ自動調整**: 画像のサイズが奇数でも、エンコードエラーが出ないように自動でパディング（調整）します。
* **軽量**: 標準ライブラリのみを使用しているため、Pythonの追加インストール（pip install）は不要です。

## 📦 必要要件

このツールを使用するには、以下の環境が必要です。

1.  **Python 3.4以上**
    * **Windows**: インストール時に "Add Python to PATH" にチェックを入れてください。
    * **macOS/Linux**: 標準でインストールされている場合が多いですが、なければインストールしてください。
2.  **FFmpeg**
    * **Windows**: 公式サイトからダウンロードし、PATHを通してください。
    * **macOS**: `brew install ffmpeg` (Homebrew使用時)
    * **Linux**: `sudo apt install ffmpeg` (Ubuntu/Debian系)

## 📂 フォルダ構成

使用する際は、1つのフォルダに以下のファイルをまとめてください。

```text
作業フォルダ/
  ├── audio2mp4.bat      <-- [Windows用] 実行バッチ
  ├── audio2mp4.py       <-- [共通] 本体スクリプト
  ├── audio.mp3          <-- [素材] 音声ファイル（名前を audio にする）
  └── cover.jpg          <-- [素材] 画像ファイル（名前を cover にする）
```

## 使い方

### 1. 素材の準備（共通）
* 動画にしたい音声ファイルを **`audio`** という名前に変更してください。（例: `audio.mp3`, `audio.wav` など）
* 背景にしたい画像ファイルを **`cover`** という名前に変更してください。（例: `cover.jpg`, `cover.png` など）
* これらをツールと同じフォルダに置きます。

### 2. 実行

#### 🪟 Windowsの場合
**`audio2mp4.bat`** をダブルクリックしてください。

#### 🍎 macOS / 🐧 Linuxの場合
ターミナルを開き、フォルダに移動して以下のコマンドを実行してください。

```bash
python3 audio2mp4.py
```

### 3. 完了
処理が終わると、同じフォルダに **`movie.mp4`** が生成されます。

## ⚙️ 設定の変更

`audio2mp4.py` をテキストエディタで開くと、上部の「設定エリア」で以下の項目を変更可能です。

```python
# ■ 設定エリア --------------------------
# 出力するファイル名
OUTPUT_FILENAME = "movie.mp4"

# 許可する拡張子リスト
AUDIO_EXTS = {'.mp3', '.wav', '.m4a', ...}
IMAGE_EXTS = {'.jpg', '.jpeg', '.png', ...}
# ---------------------------------------
```

## ⚠️ トラブルシューティング

* **「'ffmpeg' command not found」などのエラー**
    * FFmpegがインストールされていないか、PATHが通っていません。各OSのインストール方法を確認してください。
* **「必要なファイルが見つかりません」と表示される**
    * フォルダの中に `audio.xxx` または `cover.xxx` があるか確認してください。
* **Windowsでウィンドウが一瞬で消える**
    * `audio2mp4.bat` を右クリックして「編集」し、最後の行に `pause` と追記して保存してから実行すると、エラー内容が確認できます。
